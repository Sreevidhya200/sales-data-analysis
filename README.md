# sales-data-analysis
## Overview
This project analyzes sales data to identify trends, customer behavior, and business insights using Python.

## Objectives
- Analyze sales performance
- Identify top-selling products
- Find monthly revenue trends
- Visualize business insights

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Features
- Data Cleaning
- Sales Trend Analysis
- Revenue Visualization
- Product Performance Analysis

## Dataset
Sales dataset used for analysis and visualization.

## Project Structure

│
├── data/

├── README.md

## Sample Insights
- Highest sales observed during festive months
- Electronics category generated maximum revenue
- Customer retention improved repeat purchases
